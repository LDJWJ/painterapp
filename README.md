# Painter App - 이미지 편집기

클립보드에서 이미지를 붙여넣어 바로 편집할 수 있는 웹 기반 이미지 편집기입니다.
별도 설치 없이 브라우저에서 바로 사용할 수 있습니다.

## 주요 기능

| 도구 | 설명 |
|------|------|
| 선택 | 드래그로 영역을 선택합니다 |
| 사각형 | 색상과 두께를 지정해 사각형을 그립니다 |
| 화살표 | 화살표를 그려 특정 위치를 가리킵니다 |
| 모자이크 | 선택 영역에 블록 모자이크를 적용합니다 |
| 삭제 | 선택 영역을 흰색으로 지웁니다 |
| 좌클릭 / 우클릭 | 마우스 클릭 아이콘을 이미지에 삽입합니다 |
| 텍스트 | 캔버스에 텍스트를 입력하고 배치합니다 |

## 사용 방법

1. `index.html`을 브라우저에서 엽니다
2. 편집할 이미지를 복사한 뒤 **Ctrl + V** (Mac: **⌘ + V**) 로 붙여넣습니다
3. 도구를 선택하고 캔버스 위에서 드래그하여 편집합니다
4. 완료 후 **다운로드** 또는 **복사** 버튼으로 결과를 저장합니다

## 단축키

Windows와 Mac 모두 지원합니다.

| 단축키 | Mac | 동작 |
|--------|-----|------|
| `Ctrl + V` | `⌘ + V` | 이미지 붙여넣기 |
| `Ctrl + Z` | `⌘ + Z` | 실행 취소 |
| `Ctrl + C` | `⌘ + C` | 이미지 클립보드 복사 |
| `Ctrl + S` | `⌘ + S` | 이미지 다운로드 |
| `Ctrl + A` | `⌘ + A` | 전체 선택 |
| `Ctrl + =` | `⌘ + =` | 확대 |
| `Ctrl + -` | `⌘ + -` | 축소 |
| `Ctrl + 0` | `⌘ + 0` | 원래 크기 (100%) |
| `Delete` | `Delete` | 선택 영역 삭제 |

## 이미지 확대 / 축소

캔버스에 붙여넣은 이미지를 화면에서 확대하거나 축소할 수 있습니다.
실제 이미지 해상도는 유지되며, 화면 표시만 변경됩니다.

| 방법 | 동작 |
|------|------|
| 툴바 `＋` 버튼 | 확대 |
| 툴바 `－` 버튼 | 축소 |
| 툴바 `크기` 버튼 | 100%로 초기화 |
| `Ctrl / ⌘` + 마우스 휠 | 스크롤로 확대/축소 |
| 키보드 단축키 | 위 단축키 표 참조 |

## 텍스트 도구

텍스트 도구 선택 시 상단에 서식 바가 표시됩니다.

- **글꼴**: Pretendard, Noto Sans KR, 맑은 고딕, 굴림, Arial 등
- **크기**: 8px ~ 200px
- **서식**: 굵게 / 기울기 / 밑줄
- **정렬**: 왼쪽 / 가운데 / 오른쪽
- 텍스트 클릭 후 드래그로 위치 이동 가능
- `Delete` 키로 선택한 텍스트 삭제

## 옵션

- **색상**: 색상 피커 또는 프리셋(빨강, 파랑, 초록, 검정) 선택
- **두께**: 선 굵기 조절 (1px ~ 10px)
- **블록 크기**: 모자이크 블록 크기 조절 (10px ~ 30px)
