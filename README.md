# Painter App - 이미지 편집기

클립보드에서 이미지를 붙여넣어 바로 편집할 수 있는 웹 기반 이미지 편집기입니다.
별도 설치 없이 브라우저에서 바로 사용할 수 있습니다.

## 주요 기능

| 도구 | 설명 |
|------|------|
| 선택 | 드래그로 영역을 선택합니다 |
| 사각형 | 색상과 두께를 지정해 사각형을 그립니다 |
| 화살표 | 화살표를 그려 특정 위치를 가리킵니다 |
| 모자이크 | 선택 영역에 블록 모자이크를 적용합니다 |
| 삭제 | 선택 영역을 흰색으로 지웁니다 |

## 사용 방법

1. `index.html`을 브라우저에서 엽니다
2. 편집할 이미지를 복사한 뒤 **Ctrl + V** 로 붙여넣습니다
3. 도구를 선택하고 캔버스 위에서 드래그하여 편집합니다
4. 완료 후 **다운로드** 또는 **복사** 버튼으로 결과를 저장합니다

## 단축키

| 단축키 | 동작 |
|--------|------|
| `Ctrl + V` | 이미지 붙여넣기 |
| `Ctrl + Z` | 실행 취소 |
| `Ctrl + C` | 이미지 클립보드 복사 |
| `Ctrl + S` | 이미지 다운로드 |
| `Ctrl + A` | 전체 선택 |
| `Delete` | 선택 영역 삭제 |

## 옵션

- **색상**: 색상 피커 또는 프리셋(빨강, 파랑, 초록, 검정) 선택
- **두께**: 선 굵기 조절 (1px ~ 10px)
- **블록 크기**: 모자이크 블록 크기 조절 (10px ~ 30px)

## 파일 구성

```
painterapp/
├── index.html   # 메인 HTML
├── style.css    # 스타일시트
├── app.js       # 애플리케이션 로직
└── sample.png   # 샘플 이미지
```

## 기술 스택

- HTML5 Canvas API
- Vanilla JavaScript (ES6+)
- CSS3
